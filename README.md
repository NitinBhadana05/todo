# Todo

A simple TODO application / starter repository.

## Features

- Add, edit, and remove tasks
- Mark tasks as completed
- Minimal, easy-to-extend codebase

## Getting Started

These instructions will help you run the project locally.

### Prerequisites

- Node.js (>= 14) and npm, or the runtime/language used by the project

### Installation

```bash
# clone the repo
git clone https://github.com/NitinBhadana05/todo.git
dcd todo

# install dependencies (if applicable)
npm install
```

### Running

```bash
# start the app (example)
npm start
```

## Usage

Open the app in your browser (usually at http://localhost:3000) and start adding tasks.
